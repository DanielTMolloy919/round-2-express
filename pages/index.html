API Endpoints Documentation

1. Get All Items

Retrieve all items with optional filters for category and name.

Endpoint: GET /api/items

Optional Query Parameters:

  • category - Filter by category
  • name - Filter by name

Example Requests:

curl http://localhost:3010/api/items curl http://localhost:3010/api/items?category=Fruit&name=Apple

2. Create New Item

Create a new item with a name and category.

Endpoint: POST /api/items

Required Fields:

  • name - Item name
  • category - Item category

Example Request:

curl -X POST http://localhost:3010/api/items -H "Content-Type: application/json" -d '{"name": "Orange", "category": "Fruit"}'

3. Get Single Item

Retrieve a specific item by its ID.

Endpoint: GET /api/items/:id

Example Request:

curl http://localhost:3010/api/items/1

4. Update Item (Admin Only)

Update an existing item. Requires admin privileges.

Endpoint: PUT /api/items/:id

Optional Fields:

  • name - New item name
  • category - New item category

Example Request:

curl -X PUT "http://localhost:3010/api/items/1?admin=true" -H "Content-Type: application/json" -d '{"name": "Green Apple", "category": "Fruit"}'
